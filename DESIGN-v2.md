Typography\n\n### Font Pairing\n\n**Display/Heading:** [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)\n- Why: Geometric sans with distinctive letterforms (single-story 'a', angular 'G'). Feels technical and modern without being cold. Popular in developer tooling (Planetscale, Railway).\n- Variable font, weight 300-700.\n\n**Body:** [DM Sans](https://fonts.google.com/specimen/DM+Sans)\n- Why: Clean, slightly geometric, excellent readability at small sizes. Better x-height than Inter. Pairs well with Space Grotesk's geometry.\n- Variable font, weight 400-700.\n\n**Monospace:** [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)\n- Why: Industry standard for code/API keys. Ligatures for developer comfort. Retained from v1.\n\n### Font Stack\n\n```css\n--font-display: 'Space Grotesk', system-ui, sans-serif;\n--font-body: 'DM Sans', system-ui, sans-serif;\n--font-mono: 'JetBrains Mono', 'Fira Code', monospace;\n```\n\n### Loading via next/font\n\n```tsx\n// app/layout.tsx\nimport { Space_Grotesk, DM_Sans, JetBrains_Mono } from 'next/font/google'\n\nconst spaceGrotesk = Space_Grotesk({\n  subsets: ['latin'],\n  variable: '--font-display',\n  display: 'swap',\n})\nconst dmSans = DM_Sans({\n  subsets: ['latin'],\n  variable: '--font-body',\n  display: 'swap',\n})\nconst jetbrainsMono = JetBrains_Mono({\n  subsets: ['latin'],\n  variable: '--font-mono',\n  display: 'swap',\n})\n\n// <body className={`${spaceGrotesk.variable} ${dmSans.variable} ${jetbrainsMono.variable}`}>\n```\n\n### Type Scale\n\n| Token | Font | Size | Line Height | Weight | Usage |\n|-------|------|------|-------------|--------|-------|\n| display | Space Grotesk | 32px | 1.1 | 700 | Login hero title |\n| h1 | Space Grotesk | 24px | 1.2 | 600 | Page titles (\"Todos\", \"Settings\") |\n| h2 | Space Grotesk | 20px | 1.3 | 600 | Section headings |\n| h3 | Space Grotesk | 16px | 1.4 | 500 | Card titles, project names |\n| body | DM Sans | 14px | 1.5 | 400 | Todo titles, descriptions |\n| small | DM Sans | 12px | 1.5 | 400 | Metadata, timestamps |\n| caption | DM Sans | 11px | 1.4 | 500 | Badges, labels |\n| mono | JetBrains Mono | 13px | 1.5 | 400 | API keys, code |\n\n---\n\n## 3. Motion & Animation\n\n### Philosophy\nMotion serves three purposes: (1) orient the user spatially, (2) confirm actions, (3) create polish. Every animation must be under 300ms and respect `prefers-reduced-motion`.\n\n### Transition Tokens\n\n| Token | Duration | Easing | Usage |\n|-------|----------|--------|-------|\n| --duration-micro | 100ms | ease-out | Checkbox, toggle |\n| --duration-fast | 150ms | ease-out | Button hover, focus ring |\n| --duration-normal | 200ms | ease-out | Card expand, panel slide |\n| --duration-slow | 300ms | ease-in-out | Page transition, modal |\n\n### Micro-interactions\n\n| Trigger | Animation | Duration | CSS |\n|---------|-----------|----------|-----|\n| Todo complete | Checkbox scale 1 -> 1.15 -> 1 + checkmark draw | 200ms | `@keyframes check-pop` |\n| Todo complete (row) | Row opacity 1 -> 0.5, strike-through wipe left-to-right | 300ms | `@keyframes strike-wipe` |\n| Todo delete (swipe) | Row translateX(0) -> translateX(-100%) + height collapse | 250ms | `@keyframes slide-out-left` |\n| Add todo | New row height 0 -> 48px + fadeIn | 200ms | `@keyframes expand-in` |\n| FAB press | scale 0.95 -> 1 (spring) | 150ms | `transform: scale()` |\n| Page load | Staggered fadeIn+translateY(8px->0) on list items | 50ms stagger | `animation-delay: calc(var(--i) * 50ms)` |\n| Priority badge change | Color crossfade | 150ms | `transition: background-color, color` |\n| Project color swatch select | Ring expand from center | 150ms | `@keyframes ring-expand` |\n\n### Staggered List Reveal\n\n```css\n@keyframes fade-up {\n  from {\n    opacity: 0;\n    transform: translateY(8px);\n  }\n  to {\n    opacity: 1;\n    transform: translateY(0);\n  }\n}\n\n[data-stagger] > * {\n  animation: fade-up 200ms ease-out both;\n  animation-delay: calc(var(--stagger-index, 0) * 50ms);\n}\n\n@media (prefers-reduced-motion: reduce) {\n  [data-stagger] > * {\n    animation: none;\n    opacity: 1;\n  }\n}\n```\n\n### Completion Celebration\n\nWhen a root-level todo with children is fully completed (all children done):\n\n```css\n@keyframes confetti-burst {\n  0% { transform: scale(1); }\n  50% { transform: scale(1.02); }\n  100% { transform: scale(1); }\n}\n/* Subtle -- just a brief scale pulse on the row. No particle effects. */\n```\n\n---\n\n## 6.
