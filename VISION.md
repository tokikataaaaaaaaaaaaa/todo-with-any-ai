# todo-with-any-ai ビジョン

## コンセプト

どのAI（Claude, ChatGPT等）からでもMCP/APIで操作できるTodoアプリ。
単なるタスク管理ではなく、AIが「自分という人間」を理解し続けるためのパーソナルデータ基盤。

## 解決したい課題

- 既存Todoアプリはカスタマイズ性が低い（Todoistは無料5プロジェクト制限等）
- CLI/MCP操作に最適化されたTodoアプリがない
- AIに日常タスクを委任したいが、統合的な基盤がない
- 自分のコンテキストを蓄積し、AIに理解させる仕組みがない

## ターゲットユーザー

- AI開発者・CLIユーザー
- パーソナルAI基盤に興味がある人
- 生活全般（仕事・家事・育児・健康・副業）をAIで管理したい人

## 段階的ロードマップ

### 段階1（MVP）
- ツリー型構造のTodo管理（カテゴリ別アイコン付き）
- CRUD、締切、カレンダービュー
- CLI / MCP / REST API操作
- GitHub + Google ソーシャルログイン
- SP Web（モバイルファースト）
- Firebase Blazeプラン（実質無料枠内）
- Next.js (App Router) + React

### 段階2
- Claude Agent / Codex Agentを提供
- セッション中にagentを呼び出してTodoをアップデート
- 定期的に日記を自動生成・更新

### 段階3
- Second Brain連携
- 1日の行動・タスクを自動分析して日記を更新
- 改善点の提案、忘れているタスクの指摘
- 進捗・気持ちの変化をユーザーに確認

### 段階4
- スマホ/PCの利用情報をSecond Brainから取得
- 人間の1日を追跡可能に
- パーソナルAI基盤の完成形
